## ✨ 核心特性
 - 按点在球上所处的位置分配响应的频率，大小实时随响度变化

### 🚀 极致性能
- **极低性能影响** — 资源占用微乎其微
- **几乎无可见延迟** — 实时处理，感知不到延迟

### 🔄 三模式灵活切换
| 模式 | 说明 | 适用场景 |
|:---|:---|:---|
| **本地/URL 加载** | 从本地文件或网络流媒体加载音频 |
| **麦克风输入** | 实时捕获麦克风信号 |
| **窗口共享/系统内录** | 捕获特定窗口或整个系统音频 |

### ⚙️ 高自由度配置
- 广泛的配置选项，满足各类自定义需求
- 模块化设计，按需启用功能
- 灵活的参数调节，精细控制每个环节

### ❌ 音量调节要求
使用本地/URL 加载模式时，必须将系统音量调至约 1% 才能出现效果。这是由于未知bug，后续版本将优化。
